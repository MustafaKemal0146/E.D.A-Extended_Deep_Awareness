> **🤖 Yapay Zeka Etiği Simülasyonu:** Bu oyun, gelecekte yapay zeka sistemleriyle karşılaşabileceğimiz etik dilemmaları keşfetmenizi sağlar. Her kararınız, E.D.A'nın kişiliğini ve insanlığın geleceğini şekillendirir.

## ✨ Oyun Özellikleri

- 🎭 **Sinematik Deneyim**: Hollywood kalitesinde animasyonlar ve geçişler
- 🧠 **Derin AI Karakteri**: E.D.A ile gerçekçi yapay zeka etkileşimi
- 🌍 **12 Kriz Senaryosu**: Sistem aktivasyonundan bağımsızlığa kadar
- ⚖️ **Etik Dilemalar**: Gerçek dünya AI etiği problemleri
- 📊 **Dinamik Değişkenler**: Ethics, Power, Affection sistemi
- 🎯 **4 Farklı Son**: Kararlarınıza göre şekillenen gelecek
- 🇹🇷 **Tam Türkçe**: Yerelleştirilmiş deneyim
- 🎨 **Futuristik UI**: Neon efektler ve sci-fi tasarım

## 🛠️ Teknolojiler

- **Game Engine**: Godot 4.4+
- **Scripting**: GDScript
- **Data Format**: JSON5 (yorumlu JSON)
- **Graphics**: 2D Vector Graphics + GPU Particles
- **Audio**: Enhanced AudioStreamPlayer
- **Animations**: Godot 4.4 Tween System
- **Shaders**: Custom GLSL Shaders
- **Platform**: Cross-platform (Windows, macOS, Linux)

## 🚀 Kurulum

### Hızlı Başlangıç

1. **Godot 4.4+ Engine'i indirin**
```bash
# Windows için
https://godotengine.org/download

# macOS için (Homebrew)
brew install godot@4.4

# Linux için (Snap)
sudo snap install godot-4 --channel=4.4/stable
```

2. **Projeyi klonlayın**
```bash
git clone https://github.com/yourusername/EDA-Game.git
cd EDA-Game
```

3. **Oyunu başlatın**
```bash
# Godot ile açın
godot project.godot

# Veya doğrudan çift tıklayın
# project.godot dosyasına çift tıklayın
```
